Finding overlapping intervals

This project takes a list of lists with integers, i.e. intervals and returns overlapping intervals.
Here are the instructions I got:
Given a list of intervals, merge all the overlapping intervals to produce a list that has only mutually exclusive intervals.

And two example cases:

  • Input: [[1,2], [2,3], [4,5]] ----> Output: [[1,2], [4,5]]
  • Input: [[1,5], [2,3]] ----> Output: [[1,5]]
