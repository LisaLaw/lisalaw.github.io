Basic Web Crawler

This code searches a webpage for links and crawls the web link by link, with the help of Beautiful Soup.
It takes user input on the following:

  • The webpage the user wants to start on ("url")
  • Which link the user wants to use ("user_link"), i.e. the first, second, etc.
  • How many webpages deep the user wants to crawl ("user_depth")

In this case, the task was to extract a name, which is why I used a regex to find the end result.
Note: The webpage that was originally used for this problem is http://py4e-data.dr-chuck.net/known_by_Victor.html.
Note: This programme relies on having Beautiful Soup installed.

Technologies

  • Python
  • Beautiful Soup
  • RegEx
