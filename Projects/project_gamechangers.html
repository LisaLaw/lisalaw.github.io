Game Changers Mock Up

This project attempts to recreate the Game Changers webapge (gamechangersmovie.com) staying as close to their design as possible.
It has many little bits: a sign up box, image styling, buttons, anchor tags...
This project is ongoing. At the moment, I am working on the dropdown menu, please be patient until it is completely up and running. Note: At the moment, only the "getting started" and "food" tabs work. Note: All content is owned by 2019 The Game Changers Film, LLT.
