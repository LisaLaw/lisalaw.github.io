REST API NASA NeoWs

Accessing NASA's REST API 'Near Earth Object Web Service' (NeoWs).
For this project, I worked with json, accessing lists and dictionaries and extracting different data from NASA.
In the end, you can find out how mnay asteroids were registered by NASA on a given date and how many of those were hazardous.

Technologies

  • Python
  • JSON
  • Python request library
